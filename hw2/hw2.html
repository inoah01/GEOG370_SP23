My Map

This is my map for Homework 2. I have always wanted to visit Denver, CO, so I decided to create a map with a snapshot of the city.
The layer I used was a vector layer that contained points where there is public art in Denver (purple). This layer was sourced from
the Denver Open Data Catalog. In addition to this, I decided to create lines for 2 popular streets, Broadway and 16th (red); a dot to
identify Coors Field, the MLB stadium in Denver (red); and a polygon to outline a park in the area, Skyline park (green).

All data used for this project:

  • Denver Public Art Vector Data
  • GeoJSON file of the Coors Field point
  • GeoJSON file of the Skyline Park polygon
  • GeoJSON file of the Denver Streets lines
